Preparation procedure for using this feature

  • Fork the Mizar server contents repository on GitHub.
  • Set your GitHub ID and the URL of the repository you forked in step 1 on the settings page of emwiki.
  • Navigate to VS Code for the Web.
  • Install the "emvscode-web" extension.
  • Obtain an OAuth Token (a token used to authenticate access to the GitHub API).
  • Set the OAuth Token in the settings of VS Code for the Web.
